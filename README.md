# Protocol Labs - Collaborative Boundary Design

A real-time collaborative application for designing interpersonal protocols and boundaries.

## Deployment

### Backend (Node.js + Socket.io)
- **File**: `server.js`
- **Port**: Uses `process.env.PORT || 3000`
- **Dependencies**: Express, Socket.io
- **Start Command**: `node server.js`

### Frontend (React + Vite)
- **Directory**: `name-of-your-project/`
- **Build Command**: `npm run build`
- **Static Files**: `dist/` folder after build
